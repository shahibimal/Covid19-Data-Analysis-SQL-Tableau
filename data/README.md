# Data Description

This folder contains the datasets used for the COVID-19 analysis project.

## Files

### 1. CovidDeaths.xlsx
- Contains country-level COVID-19 data related to:
  - total cases
  - new cases
  - total deaths
  - population
  - continent and location
- Used to analyze infection rates, death percentages, and global trends.

### 2. CovidVaccinations.xlsx
- Contains vaccination-related data such as:
  - new vaccinations
  - total vaccinations
  - vaccination dates by country
- Used to analyze vaccination progress and population coverage.

## Data Source
- Public COVID-19 dataset (used for learning and portfolio purposes).

## Notes
- Data is stored in Excel format for easy inspection.
- SQL was used to join and analyze these datasets.
